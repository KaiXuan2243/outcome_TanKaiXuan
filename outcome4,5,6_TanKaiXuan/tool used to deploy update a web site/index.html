Describe the one tool used to deploy/update a web site

The tool used to deploy /update a web sites is FTP( File Transfer Protocol). FTP is a standard network protocol, it can transfer computer files between client and server. FTP is secured with FTPS to protect username and password. It is build on a client-server moedel. FTP has two mode that is activer mode and passive mode, it determines the estabilsment of data connection. Active mode is the client receive the data connection from server and send the FTP commmon inform server. Passive mode is client is behinf the firewall and no receive the incoming TCP connection.
