Describe the behaviours of web server and interaction with browser

Web browser is the most popular network application such as Google Chrome, Mozila Firefox, Internet Explorer and Safari. Basically it used for information browsing, but People can use it to carry out many activities such as shopping on webisite, play game and share information. While web servers are supply content for web browser, when it receive the request from browser, it find out the answer and delivers to browser, but the premise is internet network is connection. When the function of browser and web servers combine together called client server system. Client server is a standard method for design the application, the data will save in server computer and it can share to any client(computer) when the client request.
