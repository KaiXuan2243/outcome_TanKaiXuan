Describe the common methods to optimise images download size and number or requests from web server

Some common methods to optimise image download size. First is the diffent of compression, If us the lossy compression can develop the size when compress the file, and it possible can smaller than lossless compression file. Second is quality of imaga, the higher of screen resolution, better of quality of image means more of pixel and bigger of file size. Next is use suitable of image file format likes JPEG(Joint Photographic Experts Group), GIF(Graphic interchange File Format) and PNG(Portable Network Graphic). Commonly is JPEG most used.
